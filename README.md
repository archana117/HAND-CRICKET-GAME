# HAND-CRICKET-GAME
Developed an interactive Hand Cricket Game using HTML, CSS, and JavaScript.
Implemented game logic where the player competes against a computer-generated random score.
Added real-time score tracking and automatic game-over detection when both scores match.
Designed a responsive and user-friendly interface with attractive styling and animations.
Integrated a Play Again feature using a modal popup for improved user experience.
Utilized DOM manipulation, event handling, and JavaScript functions to create dynamic gameplay.
